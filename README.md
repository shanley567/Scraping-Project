## Dev Settings

### Venv

Activate the venv

1. Create the vritual environment

```powershell
python -m venv venv
```

2. Activate the venv

```powershell
.\venv\Scripts\Activate.ps1
```

The (venv) should then appear in the CLI

3. To Deactivate

```powershell
deactivate
```
